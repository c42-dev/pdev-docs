## Welcome

The pague.dev External API allows you to integrate payment processing into your applications. This documentation covers all available endpoints for managing customers, projects, charges, transactions, and PIX payments.

## Base URL

All API requests should be made to:

```
https://api.pague.dev/api/external/v1
```

## Authentication

All API endpoints require authentication using API keys. Include your credentials in the request headers:

```json
{
  "X-Account-Id": "your-account-id",
  "X-API-Key": "your-api-key"
}
```
